================================================================================
🐛 PROBLEMAS COMUNES
================================================================================

❌ "npm: command not found"
   → Instalar Node.js desde https://nodejs.org/

❌ "npm install" falla
   → Ver: README_E2E_TESTS.md → Troubleshooting

❌ Mailhog no conecta
   → Ver: VERIFICATION_CHECKLIST.md → Mailhog section
   → O correr: docker run -d -p 1025:1025 -p 8025:8025 mailhog/mailhog

❌ Supabase error
   → Verificar .env tiene SUPABASE_URL y SUPABASE_ANON_KEY reales

❌ Tests se cuelgan
   → npm run test:debug (modo debug)
   → Presionar Ctrl+C si es necesario
